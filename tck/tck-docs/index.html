Guide to Jakarta EE JSON Processing TCK 2.1 Documentation

The Jakarta EE JSON Processing TCK 2.1 documentation includes the following:

  • The Jakarta EE JSON Processing TCK Release Notes provides the information on the test harness, the test suite and the platforms tested.
    • If needed, updated Release Notes may be updated at the Jakarta EE Specification web site for Jakarta EE JSON Processing: https://jakarta.ee/specifications/jsonp/2.1/
  • The Jakarta EE JSON Processing Technology Compatibility Kit, Version 2.1 User's Guide (or PDF) provides the information that you need to install, set up, and run the Jakarta EE JSON Processing TCK, Version 2.1. In addition, the guide provides the rules you must comply with to pass the Jakarta EE JSON Processing TCK.
  • The Javadoc Assertion List lists all the javadoc assertions that are tested by the Jakarta EE JSON Processing 2.1 TCK.
